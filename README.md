# Телеграм-бот ежедневных сообщений

Бот автоматически отправляет сообщение в указанный Telegram-канал каждый день в заданное время. Сообщение содержит номер текущего дня, общее количество дней и дату.

## Требования

- Python 3.8+
- Токен Telegram-бота (получить у [@BotFather](https://t.me/BotFather))
- ID канала, в который будет отправляться сообщение

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_бота
CHANNEL_ID=ваш_id_канала
```

2. Отредактируйте `config.py` под свои нужды:

| Переменная | Описание |
|---|---|
| `START_DATE` | Дата первого дня в формате `ГГГГ-ММ-ДД` |
| `TIMEZONE` | Часовой пояс, например `"Europe/Moscow"` |
| `POST_HOUR` / `POST_MINUTE` | Время отправки сообщения |
| `TOTAL_DAYS` | Общее количество дней (знаменатель в «День X / Y») |
| `MESSAGE_TEMPLATE` | Шаблон сообщения с плейсхолдерами `{day}`, `{total}`, `{date}` |

## Запуск

```bash
# Тестовая отправка (один раз, без ожидания расписания)
python bot.py --test

# Боевой режим (работает постоянно, отправляет по расписанию)
python bot.py
```

## Структура проекта

```
bot.py           — основной код бота
config.py        — настройки (шаблон, время, часовой пояс)
requirements.txt — зависимости Python
.env             — секреты (токен и ID канала, не коммитить!)
```

## Важно

Не добавляйте файл `.env` в git. В нём хранятся секретные данные.
